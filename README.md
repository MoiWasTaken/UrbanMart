A professional, investor-ready retail analytics dashboard built with Python and Streamlit for UrbanMart, a mid-sized retail chain. This interactive web application provides comprehensive insights into sales performance, customer behavior, store operations, and profitability metrics.

---

## üìä Dashboard Preview

The dashboard provides real-time insights across multiple dimensions:

- **Executive KPIs**: Revenue, Growth, AOV, Repeat Purchase Rate
- **Store Performance**: Location-wise revenue analysis and rankings
- **Customer Insights**: Segmentation, high-value customers, loyalty metrics
- **Category Analysis**: Product performance and profitability
- **Channel Mix**: Online vs In-store revenue distribution
- **Operational Insights**: Day-wise patterns and quantity distributions
